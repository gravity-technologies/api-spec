!!! info "[ApiCrossExchVaultInvestCancelRequest](/../../schemas/api_cross_exch_vault_invest_cancel_request)"
    Request payload for canceling a pending cross-exchange vault investment.<br><br>This API allows a cross exchange vault investor to cancel a existing enqueued vault investment.<br>

    |Name<br>`Lite`|Type|Required<br>`Default`| Description |
    |-|-|-|-|
    |vault_id<br>`vi` |string|True|The unique identifier of the vault to cancel the pending investment from.|
